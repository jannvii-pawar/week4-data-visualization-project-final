# Week 4: Data Visualization & Complete Data Analysis Project

## Project Overview

This project analyzes an e-commerce sales dataset using Python, Pandas, and Matplotlib. The dataset is cleaned, analyzed, and visualized to understand sales trends, product performance, and regional sales distribution.

## Objectives

- Load and clean the dataset
- Perform exploratory data analysis
- Analyze sales data
- Create different visualizations
- Generate meaningful insights

## Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Project Structure


## Features

- Data loading
- Data cleaning
- Sales analysis
- Bar Chart
- Pie Chart
- Line Chart
- Project insights

## How to Run

1. Install the required libraries

```
pip install -r requirements.txt
```

2. Run the project

```
python main.py
```
